CLI Todo list app

I developed a simple to-do list app to get familiar with Golang. The project involved using the Cobra package to build a command-line interface for managing tasks. Tasks are stored in a JSON file, which gets updated automatically as you add or complete items. It was a hands-on way to explore the language and understand its syntax and features. This project was a valuable learning experience, helping me deepen my understanding of Golang while creating a practical tool for everyday use.

Runners Hell
